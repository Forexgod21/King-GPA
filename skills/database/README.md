# skills/database/

## Purpose

This folder supports academic database assignments within King-GPA.

It is used for planning, explanation, build support, validation, and packaging support across relational database work — including schema design, table and key planning, query planning, and writeups.

---

## How This Folder Is Used

- Supports academic database assignments end-to-end within the King-GPA workflow
- Covers planning, explanation, build support, validation, and packaging support
- Stays aligned with the shared workflow files in `pipelines/` and the core doctrine
- Does not assume Microsoft Access or other external software was actually used unless that step was completed
- Helps separate model and design work from actual software execution

---

## Files

- `skill.md` — Full definition of the database skill: supported tasks, inputs, outputs, execution logic, validation logic, and boundaries

## Rule

Keep database work structured, truthful, and easy to validate.
