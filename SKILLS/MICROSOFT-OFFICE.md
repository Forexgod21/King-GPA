# SKILL.md — Microsoft Office & Productivity Tools

## Domain
Microsoft Office suite: Word (document creation, formatting, mail merge), Excel (spreadsheets, formulas, data analysis, pivot tables), PowerPoint (presentations, slide design, speaker notes), OneNote (note organization)

## When to Use
- Document creation and formatting assignments (Word)
- Data analysis, calculations, or dashboard projects (Excel)
- Presentation delivery or slide deck assignments (PowerPoint)
- Project tracking or note-taking requirements (OneNote)
- Report writing with Word, data embedded from Excel/databases

---

## Assignment Analysis Pattern for Office Tools

### 1. **Word Document Requirements**
   - Document structure (headings, sections, TOC)
   - Formatting requirements (styles, fonts, colors, spacing)
   - Content requirements (body text, images, headers/footers)
   - Compliance requirements (APA, MLA, Chicago, etc.)
   - Page layout (margins, page breaks, orientation)
   - Special elements (tables, charts, footnotes)

### 2. **Excel Spreadsheet Requirements**
   - Data structure (rows, columns, data types)
   - Formulas and calculations needed
   - Data analysis requirements (sorting, filtering, conditional formatting)
   - Chart/graph requirements
   - Data validation rules
   - Macros or advanced features if needed

### 3. **PowerPoint Presentation Requirements**
   - Slide count and structure
   - Content per slide (text, images, videos, audio)
   - Design consistency (master slides, themes)
   - Speaker notes requirements
   - Transition and animation requirements
   - Presenter tools and delivery format

### 4. **Formatting & Style Requirements**
   - Consistent formatting across document
   - Accessibility standards (font sizes, color contrast)
   - Professional appearance
   - Branding requirements if applicable
   - File naming and organization

## Common Pitfalls in Office Tool Assignments

### Word Document Pitfalls
- ❌ Manual spacing instead of styles (breaks consistency)
- ❌ Hard returns for page breaks instead of page break tool
- ❌ Formatting applied directly instead of using styles
- ❌ Inconsistent heading hierarchy (H1 → H3 → H2 is confusing)
- ❌ Tables not properly formatted (no header row styling)
- ❌ Images not anchored (float when inserting text)
- ❌ Citation format inconsistency (APA vs. MLA mixed)
- ❌ Page numbers missing or in wrong format

**Fix:** Use Styles panel religiously. Insert page breaks, not manual returns. Define consistent citation format FIRST.

### Excel Spreadsheet Pitfalls
- ❌ Hardcoded values instead of formulas (can't update easily)
- ❌ Formulas that break when rows are inserted/deleted
- ❌ Wrong data type (text instead of numbers, prevents calculations)
- ❌ No headers on columns (data is ambiguous)
- ❌ Merged cells in data tables (breaks sorting and filtering)
- ❌ Mixing data entry with analysis in same sheet
- ❌ No data validation (garbage data causes errors)
- ❌ Color-only formatting (colorblind/printing issues)

**Fix:** Separate data from analysis. Use formulas, not values. Add data validation early.

### PowerPoint Presentation Pitfalls
- ❌ Too much text per slide (audience stops reading)
- ❌ Inconsistent design across slides (unprofessional)
- ❌ Animations that distract from content
- ❌ Low-resolution images or pixelated text
- ❌ No speaker notes (can't remember talking points)
- ❌ Fonts that are too small to read from audience
- ❌ Slides that should be 2-3 squeezed into 1
- ❌ No "thank you" slide or conclusion

**Fix:** One idea per slide. Use speaker notes extensively. Design consistency first.

## Validation Framework

### Word Document Validation
```
Requirement: "Format document in APA style"
Validation:
✅ Margins: 1 inch all sides
✅ Font: Times New Roman, 12pt
✅ Line spacing: Double-spaced
✅ Headings: Level 1 (APA format)
✅ Citations: In-text citations present
✅ Reference page: Alphabetically sorted

Status: PASS
```

### Excel Spreadsheet Validation
```
Requirement: "Calculate total sales by region"
Validation:
✅ Data properly typed (numbers not text)
✅ Regions column clearly labeled
✅ SUM formula used for totals (not hardcoded)
✅ Results match manual calculation
✅ Formula updates when data changes

Status: PASS
```

### PowerPoint Presentation Validation
```
Requirement: "10-15 slides with consistent design"
Validation:
✅ Slide count: 12 slides (within range)
✅ Master slide applied to all
✅ Fonts consistent (Calibri throughout)
✅ Color scheme consistent
✅ Images scaled appropriately
✅ Speaker notes on each slide
✅ No animation distractions

Status: PASS
```

## Code Quality Standards for Office Tools

### Word Document Standards
- Consistent use of Styles (no direct formatting)
- Proper heading hierarchy (H1 → H2, not H1 → H3)
- Clean formatting with no extra spaces
- Images properly anchored and sized
- Tables with header row styling
- Citations in required format
- No hidden formatting marks or special characters

### Excel Spreadsheet Standards
- Headers clearly labeled all columns
- Data and analysis on separate sheets (if complex)
- Formulas (not hardcoded values)
- Data validation on input cells
- Consistent number formatting
- Comments explaining complex formulas
- No merged cells in data ranges
- Professional color scheme (not rainbow)

### PowerPoint Presentation Standards
- One idea per slide (not cramped)
- Master slide templates applied consistently
- Professional fonts and color scheme
- Speaker notes on all slides
- No animation unless it adds value
- Minimal text (audience reads slides, not entire paragraphs)
- High-resolution images
- Consistent alignment and spacing

- [ ] **Content**
